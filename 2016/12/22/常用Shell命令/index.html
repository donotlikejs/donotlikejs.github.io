常用Shell命令

Posted on 2016-12-22 | In shell

本笔记操作以mac系统为准

路径表示 

./    当前目录(可省略 比如 cd ./demo 写成 cd demo)
../   上一级目录
/     绝对路径(根目录)

cd 改变目录

//进入当前目录的子目录demo
cd ./demo
//进入之前的目录
cd -

touch（新建文件）

mkdir（新建目录）

// 当前目录新建 demo.txt和index.html
touch demo.txt index.html
// 当前目录新建demo文件夹
mkdir demo

rm 删除文件

​ 常用选项

​ -i -r -v 同复制

​ -f: 忽略不存在的文件

rmdir 删除空目录

ls 显示文件列表

常用选项 -l:长格式显示结果

​ -t:按修改时间来排序

​ -a:显示所有文件 -r:相反顺序显示

​ -h:长格式列出时，不已字节显示文件大小

​ -F: 每个文件加上指示符号

​ -d: 列出指定目录中内容

pwd 显示目录路径

mv（移动/重命名文件和目录） 和 cp（拷贝文件和目录）

常用选项 -a:复制所有信息和属性

​ -i:重写前确认

​ -r:递归(复制)

​ -u:仅复制目标目录不存在或新于的文件

​ -v:显示详细命令操作信息

//复制file1文件到file2
cp file1 file2
//重命名为file2
mv file1 file2
//移动file1到根目录下的demo目录
mv file1 /demo

type 解释一个命令名

which 显示会执行哪个可执行程序（可用来找路径）

man

info

whatis
