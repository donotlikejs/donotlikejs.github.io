事件：用户或者浏览器自身执行的某种动作

事件处理程序（事件监听器）：响应某个事件的函数，事件处理程序的名字以“on”开头。

DOM0级：将一个函数赋值给一个事件处理程序属性

btn.onclick=function(){
}

DOM1级

DOM2级事件处理程序

/*
*1参数为要处理的事件名
*2参数事件处理程序的函数
*3参数为布尔值，true表示捕获阶段调用2，false表示冒泡阶段调用2
    fn在当前作用于运行
*/
addEventListener("",fn,boolean)
/*
参数同上，只是调用变成删除
*/
removeEventListener()

捕获：从不具体的父元素逐步传到具体的元素

冒泡：具体的元素逐步往上传播到不具体的父元素

DOM事件流（DOM2）:分为事件捕获阶段，处于目标阶段和事件冒泡阶段。（但浏览器都支持捕获和冒泡阶段对目标对象进行操作）

AJAX原生

  • 异步加载必须先实例化一个XMLHttpRequest函数。
let xhr = new XMLHttpRequest();
xhr.open();
xhr.send();
setRequsetHeader();
  • XMLHttpRequest的响应方法
/*
    readyStateChange 用于监听readyState的变化
    status 响应状态码 1xx 2xx 3xx 4xx 5xx
    statusText 响应状态说明
    readyState    请求／响应过程的当前活动状态
        0
        1
        2    发送，已经调用send方法
        3    接收响应数据
        4    表示完成

    responseText    响应主体返回的文本
    responseXML        响应主体返回的XML

*/
  • GET 和 POST 的一般写法
// get
let xhr=new XMLHttpRequest();
url="http://www.abc.com?a=b&c=d";
xhr.open("get",url,true);
xhr.send(null);
xhr.readyStateChange=function(){
  if(xhr.readyState==4&&xhr.status==200){
    console.log(xhr.responseText);
  }else{
    console.log(`error${xhr.status}`);
  }
};

//post
let xhr=new XMLHttpRequest();
url="http://abc.com";
data="a=b&c=d";
xhr.open("post",url,true);
xhr.setRequestHeader("Content-Type","application/x-www-from-urlencoded");
xhr.send(data);
xhr.readyStateChange=function(){
  if(xhr.readyState==4&&xhr.status=200){
    console.log(xhr.responseText);
  }else{
    console.log(`error:${xhr.status}`);

  }
}

元字符 . \b \w\d\s

/*
    .          匹配任意字符
    \b         匹配单词边界
    \w        匹配数字，字母，下划线
    \d        匹配数字
    \s        匹配空格
    [abc]    匹配方括号里任意一个字符
    ^        匹配首部
    $        匹配尾部

    --取反--
    \B \W \D \S [^abc]

*/

量词 * + ? {}

/*

    *        匹配任意个
    +        匹配一到多个
    ?        匹配 0 到 1 个
    {n}        匹配 n 个
    {n,}    匹配至少 n 个
    {n,m}    匹配 n 到 m 个

*/

分枝条件 |

/*

a|b         匹配 a 或者 b 

*/

分组 ()

/*

    (exp)            匹配且分组
    (?<name>exp)    匹配且以name名分组
    (?:exp)            只匹配不捕获不分组
    (?=exp)            匹配exp前面位置
    (?<=exp)        匹配exp后面位置
    (?!exp)            匹配后面不跟着exp的位置
    (?<!exp)        匹配前面不跟着exp的位置
    (?#comment)        注释

 */

懒惰限定符

/*

    *?
    +?
    ??
    {n,m}?
    {n,}?

*/
