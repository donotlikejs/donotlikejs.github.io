创建与删除

// 创建元素节点
document.createElement('element')
// 创建文本节点
document.createTextNode('text')
// 删除节点
element.removeChild()

获得元素

document.getElementById('idname')
element.id
document.getElementsByTagName('element')
document.getElementsByClassName('classname')
// 接收css选择符，返回与该模式匹配的第一个元素
element.querySelector()
// 返回一个NodeList()实例
element.querySelectorAll()

节点：dom的最小组成单位叫做节点（node）

html文档中节点主要有六种（一共11种） Document DocumentType Element Attribute Text DocumentFragment(文档碎片节点) comment节点（注释节点） 都继承Node对象的属性和方法

/*

    textContent //返回当前节点和它的所有后代节点的文本内容 可读写

    firstNode
