  2. 事件类型

事件：用户或者浏览器自身执行的某种动作。

事件流 页面中接收事件的顺序。

事件捕获：从不具体的父元素逐步传到具体的元素（Netscape定义的事件流）

事件冒泡：具体的元素逐步往上传播到不具体的父元素（IE定义的事件流）

DOM2级定义的DOM事件流:分为事件捕获阶段，处于目标阶段和事件冒泡阶段。（但浏览器都支持捕获和冒泡阶段对目标对象进行操作）

事件处理程序（事件监听器）：响应某个事件的函数，事件处理程序的名字以“on”开头。

事件委托（代理）：利用冒泡，只指定一个事件处理程序，就可以管理某一类型的所有事件.
