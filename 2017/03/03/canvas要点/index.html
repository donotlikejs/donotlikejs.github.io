html之canvas 绘图

发表于 2017-03-03 | 分类于 基础

使用canvas绘图共同步骤

指定canvas大小（也可称之画布大小）

调用Context(“2d”)方法

//html
<canvas id="drawing"></canvas>
<style>
#drawing{width:300px;height:300px}
</style>
<script>
let drawing=document.getElementById("drawing");
let context=drawing.Context("2d");
</script>

绘制矩形

填充:fillStyle 描边:strokeStyle

绘制矩形

fillRect(x,y,width,height)

strokeRect(x,y,width,height)

clearRect(x,y,width,height)

//填充
context.fillStyle="#e5e5e5";
context.fillRect(5,5,50,50);
//描边
context.strokeStyle="yellow";
context.strokeRect(15,15,50,50);
