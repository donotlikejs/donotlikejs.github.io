let 和 const

let和const声明都不能提升，

同一个变量不能重复声明，

声明的全局变量不是全局对象的属性

const声明和赋值同时进行，对于基本类型不能重复赋值（可以作为常量赋值），引用类型可以。

let能解决for循环中i的锁定问题。

箭头函数

箭头函数没有this，也没有argument

//let exp1=function(){return a;}
let exp1=()=>a;
//let exp2=function(){let a=0;return console.log(a);}
let exp2=()=>{let a=0;return console.log(a);}
