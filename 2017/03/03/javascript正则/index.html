修饰符

/*

    i    不区分大小
    g    全局匹配
    m    多行匹配
    u    （es6新增）

*/

元字符 . \b \w\d\s

/*
    .          匹配任意字符
    \b         匹配单词边界
    \w        匹配数字，字母，下划线
    \d        匹配数字
    \s        匹配空格
    [abc]    匹配方括号里任意一个字符
    ^        匹配首部
    $        匹配尾部

    --取反--
    \B \W \D \S [^abc]

*/

量词 * + ? {}

/*

    *        匹配任意个
    +        匹配一到多个
    ?        匹配 0 到 1 个
    {n}        匹配 n 个
    {n,}    匹配至少 n 个
    {n,m}    匹配 n 到 m 个

*/

分枝条件 |

/*

a|b         匹配 a 或者 b 

*/

分组 ()

/*

    (exp)            匹配且分组
    (?<name>exp)    匹配且以name名分组
    (?:exp)            只匹配不捕获不分组
    (?=exp)            匹配exp前面位置
    (?<=exp)        匹配exp后面位置
    (?!exp)            匹配后面不跟着exp的位置
    (?<!exp)        匹配前面不跟着exp的位置
    (?#comment)        注释

 */

懒惰限定符

/*

    *?
    +?
    ??
    {n,m}?
    {n,}?

*/

转义字符 \

/*

    需要转义的字符   * . ? + $ ^ [ ] ( ) { } / \ |

*/
