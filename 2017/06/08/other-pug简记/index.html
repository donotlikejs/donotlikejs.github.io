=和!=

= 符号会自动转义，!= 不会转义

标签

文本

p this is text
p
 this is text
p="this is text<em>自动转义</em>"
p!="this is text<em>尖括号不转义</em>"

属性

  • 属性用()包裹，属性间空格或者加逗号。如果属性名和js保留字同名需要‘’或者“”包裹。
a(class="a" href="/demo.html" 'script'="xx/1.jpg")

  • 判断

    -var istrue=true
a(class=istrue?'true':'false' href='/page.html') boolean

    class属性

  • 字面量写法

  .classname
  a.classname
  • 插入数组和合并数组
    - var classes=['a','b','c']
a(class=classes)
p(class=classes class=['newa','newb'])

  • boolean判断

    -var homeadd='/home'
a(class={active:homeadd==='/about'}) about
a(class={active:homeadd==='/home'}) home

    id属性

  • 字面量写法

    .idname
a.idname

    style属性

    style 属性值以对象字面量格式输入，css属性间用逗号间隔，css属性值为string

    a(style={border:'1px solid red',border-radio:'3px'})

    &attributes语法

    对象字面量形式

    //data-name 因为存在-，不能正常解析，需要加上双引号
p(class='a'&attributes{'data-name':'active',id:'ptext'})
- var attobj={}
- attobj.id='nav'
- attobj.herf='/page.html'
a.demo(data-name='dataname')&attributes(attobj) pagehtml

缓冲代码和不缓冲代码

不缓冲代码 前面加 - 不会直接输出

缓冲代码 前面加= 会进行HTML转义,如果不想转移，前面加!,即!=,=后面只能跟字符串或变量

block块 & extends 继承

block块

  • 当前文件使用和引用
  • include后引用其他文件的block块
  • extends后覆盖同名block或添加到同名block里

extends用来继承，且称继承与被继承文件为子模块和主模块。

  • 子模块继承主模块，如果两个模块之间无同名block块存在，子模块内容将全部被覆盖，如果两个模块之间有同名block块存在，子模块中同名block将覆盖主模块的同名block块。
  • 使用 append blockname和prepend blockname 可追加到主模块的同名block后面和前面
