% 占位符

占位符不能单独编译，需要被调用才能显示

/*%list如果不被.ul 继承，nav %list就不会被编译*/
nav %list{
  list-diretion:none;
}
.ul{
  @extend %list;
}

$ 变量符

#{} 插入

$varname:foo
p.#{$varname}{code...}

局部文件和下划线 _

scss文件前加 _ 表示局部文件，不会变编译，比如 _header.scss

@import

  • @import命令用于在当前scss文件里引入其他scss文件,可以一次@import多个文件。
  • 以下4种情况会编译为css中的@import
    • 文件包含.css
           @import demo.scss
    • 文件开头是
           @import http://donotlikejs.git.io
    • 文件名是url/(/)
    • @import里包含任何媒体查询
