##flex-box属性与值

###父容器中的属性-值

/*    
设置显示模式为flex
    display:flex;
 子项目分布方向
    flex-direction:row / column / row-reverse / column-reverse

 子项目对齐方式（主侧轴和flex-direction有关）
     主轴方向对齐（水平方向）
     justify-content:flex-start/flex-end/center/
                     space-between(项目间，项目和边界均分空白)
                     /space-around（项目间均分空白，边界空白占项目间空白的一半）
     侧轴方向对齐（垂直方向）
     align-items:stretch／center／flex-start/flex-end/baseline

 定义弹性换行方式
     flex-wrap：nowrap／wrap/wrap-reverse
 定义换行后项目的侧轴（垂直）方向对齐方式（和flex-wrap一起使用）
     align-content：stretch／center／flex-start/flex-end/
                     space-between/space-around

 flex-flow flex-direction和flex-wrap的简写模式
*/

###作用于子项目

/*
order----设置子项目的排序,数值越大越排后

重置父容器的align-items属性
align-self：auto／stretch／center/flew-start/flew-end/baseline

设置在父容器有剩余空间时，子项目的扩大比例
flex-grow

设置在父容器空间压缩时，子项目的缩小比例
flex-shrink

设置子项目被分配多余空间前，所占据的主轴空间
flex-basis

flex----flex-grow,flex-shrink和flex-basis的简写模式
*/
